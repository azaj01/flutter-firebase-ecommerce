# Flutter Xcode 15 Error (Xcode): DT_TOOLCHAIN_DIR cannot be used to evaluate LIBRARY_SEARCH_PATHS

If you're running on Xcode 15, you may encounter this issue if your Flutter project uses Firebase:

- [Issues while building iOS project with flutter](https://stackoverflow.com/questions/76791937/issues-while-building-ios-project-with-flutter/77138033#77138033)

To resolve this, **update Cocoapods to version 1.13.0 or above**.
